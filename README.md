# Objetos-en-JavaScript# Proyecto: Objetos en JavaScript – Libro

Este proyecto tiene como objetivo practicar el uso de objetos en JavaScript mediante la creación de un objeto que represente un libro.

El objeto `libro` contiene propiedades básicas como título, autor, año de publicación y estado del libro, así como métodos que permiten mostrar su información en la consola.

## Funcionalidades
- Crear un objeto libro con propiedades.
- Mostrar la información del libro mediante un método.
- Agregar y eliminar capítulos del libro.
